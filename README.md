## Installation

To get the project up and running locally, follow these steps:

1.  Clone the repository:
    ```bash
    git clone https://github.com/xhide341/lcup-poc
    ```
2.  Navigate to the project directory:
    ```bash
    cd lcup-poc
    ```
3.  Install dependencies using pnpm:
    ```bash
    pnpm install
    ```
